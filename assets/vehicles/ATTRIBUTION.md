# Vehicle sprite attribution

Vehicle sprites in this folder use the **Time Fantasy** pixel-art style by
[Jason Perry (finalbossblues)](https://finalbossblues.itch.io/).

## Shipped assets

The default `vehicle_*.bmp` files are generated by
`scripts/generate_vehicle_sprites.py` (eastbound top-down cars on a 36×22 px
canvas, per-approach color variants). They follow the palette and proportions
used in the free [Pixel Shooter and Towers Asset Pack](https://finalbossblues.itch.io/pixel-shooter-towers-asset-pack).

Regenerate after changing `VEHICLE_LENGTH` / `VEHICLE_WIDTH` in `src/config.rs`:

```bash
python3 scripts/generate_vehicle_sprites.py
```

## Replacing with pack PNGs

You may swap in PNG frames from the finalbossblues pack (or other Time Fantasy
assets). Convert to 32-bit BMP with alpha, or enable the `sdl2` `image` feature
(see root `README.md`) and name files `vehicle_south.png`, `vehicle_north.png`,
`vehicle_west.png`, `vehicle_east.png`. Sprites must be authored **facing east
(+x)**; `render.rs` rotates them with path tangent (`heading_rad`).
